# Проект "Путешествия по России"

## Описание:

Проект посвящен путешествиям и описанию природы России. Это одностраничный сайт,
который адаптирован под мобильные устройства.

### Технологии:

- HTML
- CSS

### Функциональность:

- Flexbox
- Grid
- CSS animation
- БЭМ
- Адаптивная верстка
